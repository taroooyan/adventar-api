# adventar-api

### Description
API of http://www.adventar.org

### Usage
Return data is JSON.
- title
- calendar-url
- creator
- description
- entry_count
- posted_count
- calendars
    - date
    - user
    - icon
    - comment
    - title
    - url
    - is-entry
    - is-posted

Access URL is /ARTICLE-NUMBER or /CALENDER-URL
