# adventar-api

### Description
API of http://www.adventar.org

### Usage
Return data is JSON.
```
Is_error: ,
Title: "",
Url: "",
Creator: "",
Description: "",
Entry_count: ,
Posted_count: ,
Calendars: [
    {
        Date: ,
        User: "",
        Icon: "",
        Comment: "",
        Title: "",
        Url: "",
        Is_entry: ,
        Is_posted: 
    },
    ...
]
```

Access URL is adventar/ARTICLE-NUMBER
